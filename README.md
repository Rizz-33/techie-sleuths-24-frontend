## How to run project locally ?

1. Clone the repository

```bash
git clone https://github.com/fossnsbm/techie-sleuths-24-frontend.git
```

2. Change directories

```bash
cd techie-sleuths-24-frontend
```

3. Install dependencies

```bash
yarn
```

4. Start the server

```bash
yarn dev
```
